# Policy Management System

## Description
This project is a simple Policy Management System developed using Python and Object-Oriented Programming (OOP) principles. It manages policyholders, insurance products, and payments.

## Features
- Register, suspend, and reactivate policyholders
- Create, update, suspend, and activate insurance products
- Process payments, send reminders, and apply penalties
- Demonstration with at least two policyholders

## File Structure
- policyholder.py: Manages policyholder operations
- product.py: Handles insurance product operations
- payment.py: Handles payment processing and penalties
- main.py: Demonstrates system functionality
